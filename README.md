# League Minimap champion tracker

Computer vision model made with yolov11

---

## Features

  - Can track champions on minimap and send alarms
  - Modifiable detection alarm zones
   
---

## Requirements

- Ultralytics 8.3+

- Python 3.8+

- Mss 9.0+

- Pillow 10+

- Opencv-python 4.9+

- Numpy 1.26+

Install dependencies:

```bash
InstallRequirements.bat
```
